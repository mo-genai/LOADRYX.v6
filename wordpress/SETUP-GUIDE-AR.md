## 0) ما الذي ستحصل عليه في النهاية

- موقع كامل بالعربية واتجاه RTL على دومين خاص بك (مثلاً `loadryx.com`).
- ١٤ منتج جاهز مع صور وأسعار وأوصاف.
- ٥ أقسام: PS5 AI، Setting AI، Script Xim Matrix، Accessories، الألعاب.
- سلة، خروج (Checkout)، حساب عميل، تتبع طلبات.
- دفع بمدى/فيزا/Apple Pay عبر Moyasar.
- صفحات: الرئيسية، المتجر، الأسئلة الشائعة، تواصل معنا، السياسات.

**الوقت المتوقع:** ٢–٣ ساعات للمبتدئ.

<a id="1"></a>
## 1) حجز الاستضافة والدومين

### الخيار الموصى به: **Hostinger**

1. ادخل [hostinger.com](https://www.hostinger.com).
2. اختر خطة **Premium Web Hosting** أو **Business** (الـ Business تنصح بها لمتجر).
3. اختر مدة ٤٨ شهر (أرخص شهرياً) أو ١٢ شهر لو تجرب.
4. اطلب الدومين المجاني — مثلاً `loadryx.com` أو `loadryx.store`.
5. أكمل الدفع.

> 💡 **بديل:** SiteGround (أفضل أداء، أغلى)، أو Bluehost.

### بعد الحجز

ستصلك رسالة فيها:
- بيانات الدخول للوحة hPanel
- بيانات FTP
- DNS الدومين

<a id="2"></a>
## 2) تركيب WordPress

### من hPanel:

1. ادخل لـ hPanel → اختر دومينك.
2. ابحث عن **Auto Installer** → اختر **WordPress**.
3. اضبط:
   - **اسم الموقع:** LOADRYX
   - **وصف الموقع:** مُصمَّم للاستجابة، مبني للانتصار
   - **اسم المدير (admin):** اختر اسم مستخدم قوي (لا تستخدم `admin`)
   - **كلمة مرور قوية** — احفظها في مكان آمن
   - **اللغة:** العربية
2. اضغط **Install**.

### بعد التركيب:

- ادخل لوحة التحكم عبر: `yourdomain.com/wp-admin`
- سجل بـ user/pass اللي اخترتهم.

### إعدادات أولية مهمة:

من **الإعدادات** (Settings):
1. **عام (General):**
   - عنوان الموقع: `LOADRYX`
   - الوصف: `مُصمَّم للاستجابة، مبني للانتصار`
   - المنطقة الزمنية: `Riyadh` (UTC+3)
   - تنسيق التاريخ: `Y-m-d`
2. **الروابط الدائمة (Permalinks):**
   - اختر **Post name** ← مهم لـ SEO!
3. **القراءة (Reading):**
   - اختر "صفحة ثابتة" وحدد الرئيسية لاحقاً بعد إنشاء الصفحات.

<a id="3"></a>
## 3) تركيب WooCommerce

1. من لوحة الإدارة: **الإضافات → أضف جديد**.
2. ابحث عن **WooCommerce** → اضغط **Install Now** → ثم **Activate**.
3. سيظهر معالج الإعداد:
   - **عنوان المتجر:** السعودية
   - **العملة:** Saudi Riyal (SAR)
   - **نوع المنتجات:** اختر **Digital** و **Physical** (لأن عندك الاثنين)
   - **عدد المنتجات:** أقل من ١٠٠
   - **هل تبيع في مكان آخر؟** اختر ما يناسبك
4. **تخطّى** ربط Jetpack و WooCommerce.com (مو ضروري).
5. **تخطّى** اقتراحات الإضافات حالياً — بنركّب اللي نحتاجه يدوياً.

### إضافات مهمة تركّبها الآن:

من **الإضافات → أضف جديد**:

| الإضافة | الغرض |
|---|---|
| **Loco Translate** | لترجمة أي نص باقي بالإنجليزي إلى العربية |
| **Wordfence Security** | حماية أساسية ضد الاختراق |
| **WP Mail SMTP** | لإرسال إيميلات الطلبات بنجاح |
| **WooCommerce PDF Invoices** | فواتير PDF للعملاء |

ركّب وفعّل كل واحدة.

<a id="4"></a>
## 4) تركيب قالب LOADRYX

### الخطوة A: ركّب القالب الأم Storefront

1. **المظهر → القوالب → أضف جديد**.
2. ابحث عن **Storefront** (من Automattic).
3. **Install** → **Activate**.

### الخطوة B: ركّب قالب LOADRYX الفرعي

1. حمّل الملف من الريبو: `wordpress/loadryx-child-theme.zip`
   - رابط مباشر: `https://github.com/mo-genai/LOADRYX.v6/raw/claude/sharp-gates-vvXYb/wordpress/loadryx-child-theme.zip`
2. **المظهر → القوالب → أضف جديد → ارفع قالب**.
3. اختر الملف `loadryx-child-theme.zip` → **Install Now** → **Activate**.

> ✅ الآن المتجر مفعّل بألوان LOADRYX (أزرق نيوني على خلفية داكنة) واتجاه RTL.

### الخطوة C: ارفع شعار LOADRYX

1. **المظهر → تخصيص (Customize)**.
2. **هوية الموقع (Site Identity)**:
   - ارفع شعارك (لو ما عندك واحد، استخدم اسم نصي `LOADRYX`).
   - ارفع الـ Favicon (أيقونة التبويب).
3. **اللوحة الرئيسية:**
   - **الألوان (Colors):** عدّل لون الأكسنت إلى `#0383F4` (الأزرق).
4. اضغط **Publish**.

<a id="5"></a>
## 5) استيراد المنتجات دفعة واحدة

عندك ملف `wordpress/products.csv` فيه كل المنتجات الـ ١٤ جاهزة.

### خطوات الاستيراد:

1. **WooCommerce → Products → Import** (في الأعلى).
2. اضغط **Choose File** → اختر `products.csv`.
3. اضغط **Continue**.
4. ستظهر شاشة مطابقة الحقول (Column Mapping). الأغلب سيكون مطابق تلقائياً.
   - تأكد من أن `Categories` مرتبط بـ Categories.
   - تأكد من أن `Images` مرتبط بـ Images.
5. اضغط **Run the importer**.
6. انتظر ٢–٥ دقايق (يحمل الصور من GitHub).
7. النتيجة: **"14 products imported"** ✅

### تأكد من النتيجة:

اذهب إلى **Products → All Products** — لازم تلقى:
- ٧ منتجات في PS5 AI
- ٢ في Setting AI
- ٢ في Script Xim Matrix
- ٣ في Accessories

### عدّل الأسعار/الصور لاحقاً:

أي منتج تقدر تعدّله من **Products → All Products → Edit**.

<a id="6"></a>
## 6) إنشاء الصفحات الأساسية

WooCommerce ينشئ تلقائياً: **Shop، Cart، Checkout، My Account**.

تحتاج تنشئ يدوياً:

### الصفحة الرئيسية

1. **الصفحات → أضف جديد**.
2. العنوان: `الرئيسية`
3. المحتوى:
   ```
   <h1>LOADRYX — مُصمَّم للاستجابة، مبني للانتصار</h1>
   <p>متجر سكربتات وأدوات ذكاء اصطناعي لأشهر ألعاب الشوتر التنافسية. تحسين الإيم، تثبيت الارتداد، وسرعة الاستجابة.</p>
   [products limit="8" columns="4" visibility="featured"]
   <h2>تصفّح حسب القسم</h2>
   [product_categories number="5" parent="0"]
   ```
4. **Publish**.

### صفحة الأسئلة الشائعة

1. **الصفحات → أضف جديد**.
2. العنوان: `الأسئلة الشائعة`
3. المحتوى: انسخ من `faq.html` في الريبو.
4. **Publish**.

### صفحة تواصل معنا

1. **الصفحات → أضف جديد**.
2. العنوان: `تواصل معنا`
3. المحتوى:
   ```
   <h2>تواصل معنا</h2>
   <p>للحجوزات، الاستفسارات، أو الدعم الفني — تواصل معنا مباشرة:</p>
   <ul>
     <li>📱 واتساب: <a href="https://wa.me/966573534407">+966 57 353 4407</a></li>
     <li>📞 اتصال: +966 57 353 4407 (يومياً من 10ص حتى 12م)</li>
     <li>📧 إيميل: support@loadryx.com</li>
   </ul>
   ```
4. **Publish**.

### اضبط الرئيسية:

**الإعدادات → القراءة (Reading)**:
- "صفحتك الرئيسية تعرض" → **صفحة ثابتة**
- الصفحة الرئيسية: `الرئيسية`
- صفحة المقالات: اتركها فاضية أو اعمل صفحة `Blog`.

### القائمة العلوية:

1. **المظهر → القوائم (Menus)**.
2. أنشئ قائمة جديدة باسم `Main Menu`.
3. أضف: الرئيسية، المتجر (Shop)، الأسئلة الشائعة، تواصل معنا، حسابي (My Account)، السلة (Cart).
4. تحت "Display Location" اختر **Primary Menu**.
5. **Save**.

<a id="7"></a>
## 7) إعداد بوابة الدفع السعودية (Moyasar)

> **لمَ Moyasar؟** أرخص بوابة سعودية، تدعم مدى + فيزا + Apple Pay + STC Pay، وعندها إضافة WooCommerce مجانية.

### الشروط:

- **سجل تجاري** سعودي ساري.
- حساب بنكي تجاري.
- هوية صاحب النشاط.

### الخطوات:

1. ادخل [moyasar.com](https://moyasar.com) → سجّل حساب.
2. ارفع وثائقك → انتظر الموافقة (٢–٥ أيام عمل).
3. بعد الموافقة، اطلب **API Keys** (Live + Test) من لوحة Moyasar.

### تركيب إضافة Moyasar:

1. حمّل الإضافة الرسمية من: [github.com/moyasar/moyasar-woocommerce](https://github.com/moyasar/moyasar-woocommerce)
2. **الإضافات → أضف جديد → ارفع إضافة** → اختر الملف → **Install** → **Activate**.
3. **WooCommerce → Settings → Payments**.
4. فعّل Moyasar → اضغط **Manage**.
5. أدخل:
   - **Publishable Key**: (من Moyasar)
   - **Secret Key**: (من Moyasar)
   - فعّل **مدى**, **Credit Card**, **Apple Pay**, **STC Pay**.
6. **Save**.

### بديل: HyperPay

نفس الفكرة، لكن أغلى. اتبع نفس الخطوات من موقع [hyperpay.com](https://www.hyperpay.com).

### قبل ما تستلم دفعات حقيقية:

- جرّب طلب وهمي في **وضع Test** أولاً.
- تأكد من إرسال إيميل التأكيد للعميل.

<a id="8"></a>
## 8) إعدادات الشحن (للمنتجات الفعلية)

المنتجات الفعلية عندك: **Titan Two, Brook Wingman, P5 General**.

### الخطوات:

1. **WooCommerce → Settings → Shipping**.
2. **Add shipping zone** → سمّه `السعودية`.
3. Zone regions: **Saudi Arabia**.
4. **Add shipping method**:
   - **Flat rate**: ٢٥ ريال داخل المدن الرئيسية.
   - **Local pickup**: ٠ ريال (استلام من المتجر).
5. **Save**.

### لو تبي شركة شحن أتمتة:

ركّب إضافة **Aramex** أو **SMSA** من WooCommerce Extensions.

<a id="9"></a>
## 9) إعداد المنتجات الرقمية (السكربتات والـ AI)

المنتجات الرقمية: PS5 AI (٧ منتجات)، Setting AI (٢)، Script (٢).

في الـ CSV هذي المنتجات معلّمة `Virtual=1` تلقائياً، يعني:
- ما تحتاج عنوان شحن من العميل.
- ما يحسبها في رسوم الشحن.

### بعد ما يطلب العميل:

النموذج الحالي = **تواصل يدوي**:
1. يصلك إيميل بالطلب.
2. تتواصل مع العميل عبر WhatsApp/إيميل لجدولة موعد الإعداد.
3. تنفّذ الخدمة.
4. تحدّث حالة الطلب من **Processing → Completed**.

### لو تبي ملفات تحميل تلقائي (مثلاً ملف سكربت):

في صفحة المنتج → **Product Data** → فعّل **Downloadable** → ارفع الملف.

<a id="10"></a>
## 10) قائمة فحص قبل النشر

- [ ] لوحة الإدارة مؤمّنة بكلمة مرور قوية + Wordfence فعّال
- [ ] شهادة SSL مفعّلة (الـ HTTPS يشتغل) — Hostinger يفعّلها تلقائياً
- [ ] العملة = SAR، اللغة = العربية، التوقيت = الرياض
- [ ] كل المنتجات لها صورة + سعر + وصف
- [ ] صفحات السياسات منشورة: الخصوصية، الشروط، الاسترجاع
- [ ] بوابة الدفع Live (موش Test)
- [ ] جرّبت طلب وهمي من البداية للنهاية
- [ ] إيميل الإدارة يستقبل إشعارات الطلبات
- [ ] السلة + الخروج يشتغلون على الموبايل
- [ ] القائمة العلوية تحتوي على: الرئيسية، المتجر، حسابي، السلة، تواصل
- [ ] WhatsApp زر تواصل ظاهر في الموقع
- [ ] Google Analytics + Search Console مربوطين
- [ ] نسخة احتياطية مفعّلة (UpdraftPlus)

<a id="11"></a>
## 11) نصائح ومشاكل شائعة

### الصور ما طلعت بعد الاستيراد؟

سبب شائع: الصور تحاول تحميل من GitHub والـ branch مو متاح. الحلول:
- تأكد من أن الفرع `main` يحتوي الصور (أو غيّر الرابط في الـ CSV قبل الاستيراد).
- بديل: ارفع الصور يدوياً من **Media → Add New**، ثم اربطها بكل منتج.

### الموقع بطيء؟

- ركّب إضافة كاش: **LiteSpeed Cache** (لو على Hostinger) أو **WP Rocket**.
- ضغط الصور: **Smush** أو **ShortPixel**.

### العربية تطلع مكسّرة في الإيميلات؟

- ركّب **WP Mail SMTP** واربطه بـ Gmail/SendGrid.

### كيف أضيف منتج جديد بعدين؟

**Products → Add New** → عبّي الاسم، السعر، الوصف، الصورة، القسم → **Publish**.

### مشكلة في الـ Checkout؟

- تأكد من تفعيل العملة SAR بشكل صحيح.
- تأكد من أن بوابة الدفع في وضع **Live** (مو Test).

### دعم مباشر:

- WooCommerce: [docs.woocommerce.com](https://docs.woocommerce.com)
- Storefront: [woocommerce.com/storefront](https://woocommerce.com/storefront)
- Moyasar: [docs.moyasar.com](https://docs.moyasar.com)
